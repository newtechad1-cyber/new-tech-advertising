## Mission

> "Do not build another feature until the existing Knowledge System becomes measurable."

R0.7 transforms the NTA Operating System from a content management system into a **business intelligence platform**. The question is no longer *"What can it do?"* — it's *"How do we make sure the world discovers it?"*

---

## Phases

### Phase 1 — AI Crawl Optimization ✅
**Priority:** CEO #1 priority

#### 1a. AI Visibility Layer
- `/public/llms.txt` — LLM-friendly site descriptor following the emerging `llms.txt` standard
- `/public/ai-sitemap.json` — Machine-readable content graph with entity relationships, topic clusters, article metadata, and collection structure
- `robots.txt` updated to expose AI files

#### 1b. Expanded Structured Data
Enhanced `SEOHead.jsx` with 11 JSON-LD schema types:
| Schema | Purpose |
|--------|---------|
| WebSite + SearchAction | Sitewide entity recognition |
| Person (Rick Hesse) | Founder authority with @id |
| ProfessionalService | Organization with services catalog |
| Article | Canon articles and blog posts |
| VideoObject | YouTube video metadata |
| LearningResource | Learning Center content |
| CollectionPage | Canon collections and reader journeys |
| HowTo | Growth tools and discovery processes |
| FAQPage | Structured Q&A |
| BreadcrumbList | Navigation hierarchy |
| Speakable | Voice assistant compatibility |

#### 1c. Topical Authority Map
- 8 topic clusters defined in `src/data/topicClusters.js`
- Each cluster: pillar page + supporting content + keywords + buyer stages
- Visual component: `TopicalAuthorityMap.jsx`
- Internal linking suggestion engine: `getInternalLinkSuggestions()`

**Clusters:**
1. AI Search & Visibility
2. Digital Trust & Authority
3. AI Marketing for Local Business
4. Content & Video Strategy
5. Small Business Growth Systems
6. The Founder Journey
7. HVAC Marketing
8. Plumbing Marketing

### Phase 2 — Internal Authority Engine (Planned)
- Page scoring: authority, link score, completion rate, collection depth, citation count, video connections
- Automated recommendation engine

### Phase 3 — Build Logs Become Content (Planned)
- Every release auto-generates: Journal article, YouTube script, LinkedIn, Facebook, X, Instagram, TikTok, Newsletter, Learning Center lesson
- "Releases" → "Build Logs" public rebrand

### Phase 4 — Authority Dashboard ✅
Route: `/admin/authority`

5-tab dashboard answering the CEO's questions:
| Tab | Answers |
|-----|---------|
| Overview | Content composition, buyer stage coverage, cluster depth |
| Google | Search Console data: indexed, not indexed, impressions, CTR, top pages |
| AI Visibility | Schema readiness, AI sitemaps, entity recognition status |
| Topic Authority | Full topical authority map with cluster details |
| Gaps & Actions | Orphaned pages, weak stages, incomplete topics, "What should Rick publish next?" |

### Phase 5 — Canon Completion (Planned)
- Expand beyond initial 96 assets / 21 articles / 13 videos
- Include years of blog posts, Learning Center material, older videos, documents
